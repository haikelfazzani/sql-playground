# Siql
Free and open source online SQL playground

### Keys
- Run Code: Ctrl + Enter
- Beautify Code: Ctrl + Alt + F
- Clear Code: Ctrl + Alt + C

### Built with
- codemirror / react-codemirror2
- react
- react-split
- sql-formatter
- sql.js
- Backend: Lumen (Laravel) as restful service (Pastebin API)
